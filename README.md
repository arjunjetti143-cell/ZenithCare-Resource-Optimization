# ZenithCare Resource Optimization

Healthcare analytics project focused on patient flow optimization, bed occupancy analysis, staff allocation, and length of stay prediction using SQL, Python, Power BI, and Machine Learning.

## Objectives
- Analyze peak admission hours
- Calculate Average Length of Stay (ALOS)
- Monitor bed occupancy
- Optimize staff-to-patient ratios
- Predict patient stay duration using Machine Learning

## Tools
- SQL
- Python
- Power BI
- PostgreSQL / SQLite
- Random Forest
